Sencha ExtJs

本次项目的主系统使用了ExtJs，他是一个前端JS框架是本次实训的重点，其表格控件功能丰富、界面美观、轻量级的实现，都是其作为大型框架的优势。

Vue.js

Vue.js是一套构建用户界面的渐进式框架。他只关注视图层，采用自底向上增量开发的设计。项目中，销售系统界面的实现则是使用Vue.js。

JQuery

JQuery是一个快速、简洁的JS框架。除了主系统的ExtJs，和销售系统的Vue.js外，JQuery作为辅助实现登陆页面、错误页面等。
